`https://little-url.glitch.me/new/https://www.google.com`

`{ "original_url":"https://www.google.com" ,"short_url":"https://little-url.glitch.me/1632447086" }`

### Usage:

`https://little-url.glitch.me/1632447086`

### Will redirect to:

`https://www.google.com`
